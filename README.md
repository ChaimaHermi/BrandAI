BrandAI est une plateforme automatisée qui permet aux entrepreneurs de transformer
leur idée de startup en identité de marque complète grâce à un pipeline multi-agents IA.

À partir d'une simple idée, BrandAI génère automatiquement :

- 🔍 Analyse de marché et de la concurrence
- 🎨 Identité de marque (nom, slogan, logo, couleurs)
- 📱 Contenu pour les réseaux sociaux
- 🌐 Site web vitrine
- 📈 Stratégie marketing et recommandations
